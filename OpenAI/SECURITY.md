# Security and Responsible Use

## Scope

This repository is a public Codex skill for lawful editorial work on original or authorized text.

It is not intended for:

- illegal or abusive content generation
- sexual content involving minors
- coercive sexual abuse content
- hate, extremist praise, fraud, forgery, or violent wrongdoing assistance
- copyrighted-text replication beyond limited analysis use
- living-author imitation intended to substitute for that author

## Reporting

If you discover:

- a script behavior that materially conflicts with the documented safety boundaries
- a bundled reference that creates avoidable legal or licensing risk
- a packaging mistake such as third-party copyrighted material or font binaries

open an issue in the repository or remove the repository from public distribution until it is corrected.

## Maintenance expectations

- Re-check all external font-license references before each public release.
- Do not merge private client material, unpublished manuscripts, or copyrighted third-party text into this repo.
- Keep refusal boundaries and compliance notes aligned with the actual skill behavior.
- Treat all compliance output as screening support, not legal advice.

## No warranty

This repository includes screening helpers, not guarantees.

Publication legality, platform compliance, copyright clearance, and font licensing still require human review in the relevant jurisdiction and release context.
