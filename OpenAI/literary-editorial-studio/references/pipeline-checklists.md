# Pipeline Checklists

## Keep everything inside `literary-editorial-studio` when the request combines

- project setup plus chapter planning
- continuation plus rewrite
- rewrite plus manuscript cleanup
- manuscript cleanup plus release packet
- manuscript cleanup plus `.docx` handoff
- editorial revision plus compliance screening
- typography planning plus publication packaging

## Fast routing

### Architecture

- premise
- intended readership
- canon or continuity anchors
- chapter or section map
- next-step target

### Development

- structural weakness
- pacing drift
- clarity gaps
- payoff timing
- continuity-sensitive areas

### Prose

- scope
- edit depth
- forbidden changes
- diction or voice goal
- terminology to preserve

### Manuscript

- source of truth
- heading hierarchy
- numbering scheme
- support docs needed
- delivery target

### Compliance

- flagged passage
- risk category
- safer rewrite path
- final manual review needed
