# Manuscript Presets

Use the lightest preset that solves the user's problem.

## Draft Master

```markdown
# Project Title
- Status:
- Scope:
- Last revised:

## Part 01: Part Title

### Chapter 001: Chapter Title

[chapter text]

## Appendix A: Cast Sheet

- Name:
  - Role:
  - Current status:
```

## Folder Split

```text
project-root/
  00-project-frame.md
  01-story-bible.md
  02-continuity-ledger.md
  manuscript/
    001-chapter-title.md
    002-chapter-title.md
  support/
    synopsis.md
    cast-sheet.md
    glossary.md
```

Use this when one file becomes hard to review or maintain.

## QA Checklist

- Heading levels are stable.
- Numbering is continuous.
- Terminology is consistent.
- Support docs do not contradict the manuscript.
- Delivery notes match the actual target format.
